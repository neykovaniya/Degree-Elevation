# **Повишаване на степента**

Ния Нейкова, СИ, 2 курс

## **Три основни концепции**

**Какво е крива на Безие?**
Параметрична крива, използвана в компютърната графика за моделиране на гладки форми.
**Алгоритъмът на De Casteljau**
Числен метод за намиране на точка върху кривата за дадено t.
**Повишаване на степента (Degree Elevation)**
Процес на преобразуване на крива от степен n в крива от степен n+1.



## **Цел на проекта**

- **Практическа реализация** на алгоритъма на De Casteljau за построяване и Degree Elevation за повишаване на степента
- **Показване** как увеличаването на контролните точки променя контролния полигон, но запазва формата на кривата
- **Създаване** на интуитивна среда за експериментиране с математически концепции от компютърната графика



## **Използвани технологии**

**Core Logic: Vanilla JavaScript (ES6+)**

Без използване на тежки библиотеки и обектно-ориентиран подход за управление на състоянието

**Визуализация: HTML5 Canvas API (2D Context)**

Използван за високопроизводително чертаене на кривите и полигоните

**Интерфейс: CSS3 (Grid, Flexbox & CSS Variables)**

Responsive Design за мобилни и десктоп устройства

**Математически формули: KaTeX**
